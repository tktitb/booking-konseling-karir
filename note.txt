	Aplikasi Booking Konseling Karir â€“ ITB Career Center

1. Pendahuluan
- Tujuan: Membuat aplikasi web untuk memudahkan peserta melakukan booking konseling karir dengan psikolog ITB Career Center.
- Lingkup: Sistem hanya digunakan oleh peserta dan admin. Psikolog tidak perlu login.
- Masalah saat ini: Proses booking dilakukan manual via WhatsApp, pencatatan di Google Spreadsheet.
- Solusi: Web interaktif dengan integrasi Google Sheets API + WhatsApp link generator.

2. Aktor Sistem
- Peserta
- Melihat jadwal konseling (kalender bulanan, mingguan, harian).
- Mengisi form booking.
- Mengirim request ke WhatsApp ITB Career Center.
- Admin
- Menerima request via WhatsApp.
- Validasi data.
- Update status di Google Spreadsheet.

3. Fitur Utama
3.1 Kalender Jadwal
- Menampilkan jadwal konseling per bulan, minggu, hari.
- Data diambil dari Google Spreadsheet (sheet per bulan).
- Slot ditandai: Kosong (bisa dibooking) / Terisi (tidak bisa dibooking).
3.2 Form Booking
- Input:
- Nama lengkap
- Sesi konseling (Online/Offline)
- Kategori peserta (Mahasiswa ITB, Alumni ITB, Mahasiswa Non-ITB, Alumni Non-ITB)
- Pilih jadwal konseling (validasi slot kosong)
- Nomor HP
- Email
- Output: Generate pesan otomatis ke WhatsApp ITB Career Center.
3.3 Validasi
- Slot yang sudah terisi tidak bisa dipilih.
- Form tidak bisa submit jika ada data kosong.
3.4 Integrasi WhatsApp
- Submit form â†’ generate pesan otomatis sesuai format:

Hari & Waktu Konseling: [Tanggal & Jam]
No. HP (WhatsApp): [Nomor]
Email: [Email]


3.5 Konten Informasi
- Foto psikolog.
- Ajakan konseling (CTA).
- Informasi lokasi offline (ITB Career Center, GKU Timur, Lantai Dasar ITB Ganesha).
- Kontak: Instagram, Website, WhatsApp.

4. Data & Integrasi
- Google Spreadsheet:
- Kolom A: Tanggal & Waktu
- Kolom B: Psikolog
- Kolom C: Mode (Online/Offline)
- Kolom F: Status (Terpenuhi/Belum dimulai)
- Nama Sheet: Januari, Februari, dst.
- API: Google Sheets API untuk query data sesuai bulan.

5. Tampilan & UI/UX
- Modern, menarik, responsif, animatif.
- Warna utama:
- #F3F3E0 (light background)
- #27548A (primary blue)
- #183B4E (dark navy)
- #DDA853 (accent gold)
- Kalender interaktif dengan highlight slot kosong/terisi.
- Efek hover, transisi smooth, CTA jelas.

6. Requirement Non-Fungsional
- Performance: Jadwal harus tampil cepat (real-time query ke Google Sheets).
- Security: Validasi input form, sanitasi data.
- Scalability: Struktur siap untuk penambahan psikolog/jadwal di masa depan.
- Maintainability: Integrasi API modular, mudah diupdate.
- Usability: Tampilan sederhana, mudah dipahami peserta umum.
